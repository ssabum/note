## 총평

> 비교적 길지 않은 코드로 재밌는 영상을 만들 수 있어서 재밌었다.
>

>
> 특이한 이펙트의 배경영상이 준비되어 있다면 다양한 효과를 연출하는 실시간 webcam을 제작할 수 있을 것이다.

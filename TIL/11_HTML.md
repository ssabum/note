## 기본 Tag

```python
<strong></strong> : 굵게
<u></u> : 밑줄
<h1></h1> : 제목
<li></li> : 리스트
<ul></ul> : 그룹핑 리스트
<ol></ol>: 순서있는 그룹핑 리스트
<select></select>: 목록 리스트
<textarea></textarea>: 설명
```

## Form

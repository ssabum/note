# Karaoke Microphone

> `Pyaudio`를 이용해서 목소리를 입력받아 에코를 넣어서 출력해보자

## Features

- Voice echo
- Audio delay

## Dependency

- pyaudio https://people.csail.mit.edu/hubert/pyaudio
- numpy

## 총평
